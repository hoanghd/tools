- Ansible: Giúp điều khiển máy chủ từ xa để cài đặt, cập nhật soft
  ## Ubuntu
  sudo apt install ansible nfs-kernel-server
  ## Mac
  pip install ansible
  
- Vagrant: Để tạo ra các máy ảo băng lệnh
  + https://www.vagrantup.com/downloads.html #Download
  + vagrant -h #Check system
  + https://vagrantcloud.com/discover/featured

- Create
  $ mkdir /vagrant/hoanghd.local
  $ vagrant init
    Hoặc có thể sử dụng 2 config file sau (Vagrantfile, MyVagrantfile)
  $ Copy keys & /vagrant/hoanghd.local/keys
    
  $ vagrant box add --force hoanghd.local hoanghd.local.box
    * Download box từ URL https://vagrantcloud.com/discover/featured đổi tên thành `hoanghd.local.box`
    * Hoặc download từ https://drive.google.com/open?id=0B_7iXI5IZTlXSEJhVXQyUDJyMFU đổi thành `hoanghd.local.box`
    
  $ vagrant up --no-provision
  $ sudo sh -c "echo '192.168.35.10 hoanghd.local' >> /etc/hosts"
    Chú ý IP: 192.168.xx.xx
  $ vagrant ssh hoặc ssh vagrant@hoanghd.local #pw: vagrant
  
  $ vagrant provision
  
  $ vagrant halt  #Stop
  $ vagrant destroy #Delete

- Sample
  $ mkdir /vagrant/hoanghd.local
  $ cp ~/Documents/base.local.box hoanghd.local.box
  $ vagrant init
    Edit File Vagrantfile
    -------------------------------------------------
    Vagrant.configure(2) do |config|
      host = "hoanghd.local"
      ip   = "192.168.35.10"
      port = 2224
    
      config.ssh.username = "vagrant"
      config.ssh.password = "vagrant"
      config.vm.box = "#{host}"
      config.vm.network "private_network", ip: "#{ip}"
      config.vm.network "forwarded_port", guest: 22, host: port, id: "ssh"	
    
      config.vm.provider :virtualbox do |vb|
        vb.name = "#{host}"
        vb.customize ["modifyvm", :id, "--memory", "1024", "--cpus", "2"]
        vb.customize ["modifyvm", :id, "--hwvirtex", "on"]
        vb.customize ["modifyvm", :id, "--audio", "none"]
        vb.customize ["modifyvm", :id, "--nictype1", "virtio"]
        vb.customize ["modifyvm", :id, "--nictype2", "virtio"]
      end
        config.vm.define "#{host}" do |t|
        t.vm.hostname = "#{host}"
      end
    end
